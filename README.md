GPA Calculator (C++ Project)
This is a C++ console project that calculates GPA based on subjects, credit hours, and grades.
It uses loops, arrays, and conditional statements.
- Add multiple subjects
- Enter credit hours and grades
- GPA calculate automatically
- Simple console output
- GPA = (Grade Point × Credit Hours) / Total Credit Hours
- 1. Open file in C++
2. Compile program
3. Run it
